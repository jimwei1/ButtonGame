READ ME:

1. Rules:

After the game is started, the time will activate. Press the button anytime while the timer is running. The first and last player who press the button are eliminated. Two-way ties broken by whomever presses the button last. Last one alive wins.

2. How to play:

Minimum 3 players. Once enough players join the lobby, any player can start the game by pressing "Start button" twice. I'm not sure why it has to be pressed twice. Please only one person press the start button.
